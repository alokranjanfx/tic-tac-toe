# Tic Tac Toe 🎮

A clean, responsive Tic Tac Toe game built with **Next.js** and **React**.

## Features

- Two-player local gameplay (X and O)
- Win and draw detection with highlighted winning line
- Persistent scoreboard (wins per player + draws)
- "New Round" and "Reset Scores" controls
- Responsive, mobile-friendly design

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for production

```bash
npm run build
npm start
```

## Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
